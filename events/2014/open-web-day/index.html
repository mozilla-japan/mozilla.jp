Mozilla Open Web Day in Tokyo へのご来場ありがとうございました

最新情報

  • 10 / 03
    イベント当日 10/05 の 13:00 から、ステージプログラムを Ustream でインターネット配信することが決定しました。
  • 09 / 26
    ウェブサイトを公開しました。

Mozilla Open Web Day in Tokyoは、Mozilla に関係する東西さまざまなコミュニティや研究プロジェクトが集結し、その成果の展示・発表を行う収穫祭的なイベントです。

モバイルデバイスの新たなプラットフォームとして期待される Firefox OS に関するプロジェクトをはじめ、Web of Things や Personal Fabrication、Linked Open Data、Firefox の最新技術、ローカライズ、アドオンなど、Mozilla を取り巻く最新の技術や思想のバザールです。各グループの展示やオープンステージでの発表、プロジェクト同士の交流を介し、ウェブや Open の可能性について考える一日、それが Mozilla Open Web Day in Tokyo です。

  • FabNavi: ものづくりの過程を記録／共有／再生する次世代ファブ基盤
    FabNavi プロジェクト
    渡邊恵太 (明治大学総合数理学部), 塚田浩二 (公立はこだて未来大学), 沖真帆 (公立はこだて未来大学), 赤塚大典 (Mozilla Japan)
    ウェブサイト
    http://fabnavi.org/

    近年、3D プリンターなどのデジタル工作機械や、それらで出力可能なデータをウェブ上で共有するサービスが広がり、各家庭でエンドユーザー自身がものづくりに携わるデジタル・ファブリケーション時代の到来が期待されています。こうした環境では様々な「部品」の製作が容易になる一方、それらの人手での組み立てを支援するシステムは十分に提案されてきませんでした。

    FabNavi では、実世界でのアセンブリ過程 (例： 電子回路の実装，3D プリンターで出力した樹脂部品の組み立て等) を手軽に記録／編集し、標準化された形式でウェブ上で共有、多様な環境で再生可能とすることで、デジタル・ファブリケーション時代のものづくりの基盤となるシステムの構築を目指します。

    『Foxroid Tips』は、Firefox OS アプリの実装方法を、Android アプリのコードと比較しながら知ることができる Tips 集です。

    Android デベロッパーはもちろんのこと、これから Firefox OS アプリを作成しようとするすべての開発者の方にご活用頂けます。

  • ウェブで音と遊ぶ
    W3C (World Wide Web Consortium)
    ウェブサイト
    http://www.w3.org/

    Web Audio API はウェブアプリケーションに実装されている JavaScript API です。ウェブ開発者は、サクサク動作するオーディオプロセッシングの機能を用いてブラウザー上でオーディオの作成・変更・分析を行なうことができます。加えて Web Audio API は、別々のプラットフォームやブラウザーの垣根を越えた仕様を搭載しており、特定のプラグインやソフトウェアに依存せずに開発することを可能にしています。今後は教育、研究、デジタルパブリッシングやエンターテイメント分野などで更なる活用が見込まれています。

  • 中央大学オープンプロジェクト演習
    中央大学オープンプロジェクト演習有志メンバー
    豊田将平, 森下拓也, 渡邊千尋

    中央大学理工学部情報工学科の「オープンプロジェクト演習」という科目の成果物の展示を行います。この科目では、昨今の "Internet of Things" の広まりを背景に、ウェブとハードウェアを繋げた新たなデバイスの製作を行いました。各々でアイデアを出し、実装には Arduino といったオープンハードウェアやオープンソースソフトウェアを活用した "オープン" なプロジェクトとして開発を進めていきました。

  • 北国からの Firefox OS
    公立はこだて未来大学 塚田研究室
    ウェブサイト
    http://tsuka-lab.org/

    今回は 2014 年 7 月にはこだて未来大で行われた Firefox OS ワークショップの内容を発展させた 2 つのプロジェクトを展示・紹介します。一つ目は、「雨男・雨女」または「晴れ男・晴れ女」を判定するアプリ「晴れ時々雨People」。日々の生活の中で、外出時の天気の状態を端末側で自動的に調べ、個人のデータとして蓄積することでユーザーが雨女か、晴れ女であるかをレベル分けして判定・表示して、SNS に投稿するゲーム感覚アプリです。二つ目は、「PuTalk!」。スマートフォンを耳に当てると通常の 1:1 通話を、机に置くとスピーカーフォンに自動切り替えして、 N:N のグループ通話を実現します。

  • 夏の Firefox OS アプリ自由研究
    夏の Firefox OS アプリ自由研究会
    シュン, 若月舞子, Rui, 川上治美
    ウェブサイト
    http://webmaker.mozillafactory.org/post/96436564487

    夏休みの期間に開催した「Maker Party」では、小学生から大学生までの学生が集まり、アプリを開発しました。この企画では、中央大学の学生に協力してもらい、「Firefox OS 端末のセンサーを利用したアプリ」という制作テーマにのっとって、各々の自由な感性に任せたアプリ開発を行いました。「Firefox OS」のアプリ開発には、沢山の可能性が詰まっており、そこに携われたことで、私達も楽しい時間を過ごすことができました。ご来場お待ちしております。

  • 学び × FirefoxOS
    関西大学総合情報学部松下研究室
    ウェブサイト
    http://amateras.wsd.kutc.kansai-u.ac.jp/?p=2047

    関西大学総合情報学部 松下研究室で行っている研究のうち、Firefox OS 上でを３つ紹介します。一つ目は「Word Scrap」。これは昨年度の FirefoxOS ハッカソンで開発したアプリの改良版です。二つ目は「冒険うぃる! mk2」。今年のウェブハッカソンで開発した街歩きアプリを歴史の学びに拡張しています。そして三つ目は「オノマトペ絵本」。外国人にわかりにくいオノマトペのニュアンスを、絵本を通じて学んでもらうアプリです。

  • 端末試用会、アイロンビーズ × NFC ワークショップ、AppMaker デモ
    Firefox OS コミュニティ
    ウェブサイト
    http://fxos.org/

    Firefox OS コミュニティには、Firefox OS に興味のある有志が集まっています。デザイナーやエンジニアはもちろん、誰もがそれぞれにできることを見つけて Firefox OS に関して日々研究をし、みんなで楽しみながら Firefox OS に貢献しています。

    Mozilla Open Web Day in Tokyo では、端末の試用会のほか、NFCを組み合わせたアイロンビーズのワークショップ、ブラウザー上で簡単にアプリを作れるAppMakerのデモなどを行います。Firefox OS の魅力と、コミュニティの活動の雰囲気が伝われば幸いです。

  • 3DVR 体験 - タオバイザー
    タオソフトウェア株式会社
    ウェブサイト
    http://www.taovisor.com

    Google が開発者イベントで cardboard というダンボール製の 3D ビュアーを配布しました。とても面白いのですが、3D は体験してみないことに は、その面白さをわかってもらうことができません。そこで cardboard 互換のタオバイザーを作成しました。Firefox OS では WebGL を使用する事で 3DVR の世界を楽しむ事ができます。タオバイザーと Firefox の組み合わせで 3DVR の体験デモを行います。
