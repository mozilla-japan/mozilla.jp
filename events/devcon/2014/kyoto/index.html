Firefox Developers Conference 2014 in Kyoto  Firefox OS で解き放て！未来

開催主旨

Mozilla Japan 主催の Firefox Developers Conference を 4 月 19 日 (土) に京都で開催します。今回のテーマは、オープンなウェブ標準技術に基づくアプリケーションプラットフォームとして、今注目の「Firefox OS」。

基調講演には、米国 Mozilla Corporation で Firefox Marketplace / アプリ担当の Vice President を務める Rick Fant が来日するほか、日本における製品展開も見据えつつ Firefox OS の未来をお伝えします。アプリ開発の基本から最新の開発環境のご紹介、国内コンテンツプロバイダーによるトークセッションなど、多彩なゲストをお招きして Firefox OS のすべてを語り尽くします！

あらゆるデバイスがウェブに繋がりウェブ技術だけで何でもできる時代を前に、アプリやデバイスの開発者はもちろん、ウェブデザイナーやウェブ開発者はこれまでの知識と経験を活かして新しい領域に踏み出すきっかけにしていただければと思います。

Firefox OS で創る新しい世界

登壇者
  • Mozilla Japan モバイル＆エコシステム マネージャー 浅井 智也
概要

Firefox OS では UI やアプリを作るウェブ技術だけでなく開発プロセスやロードマップもオープンにして、世界中のパートナー企業が Mozilla と共にユーザーの求める価格・機能・安心・ユーザー体験を実現する製品の開発を続けています。日本国内においてもスマートフォンやスマートテレビなどの開発が発表され、コンテンツプロバイダー各社による対応も進められています。

Firefox OS はこれから何処に向かうのか。発売予定の端末、Firefox OS 2.0 に向けて開発が進められている UI や機能はもちろん、Mozilla はパートナー各社とどのような世界を創っていこうとしているのかご紹介します。

できる！スマホアプリ：ウェブからはじまるアプリ開発

登壇者
  • 株式会社グローバルサイバーグループ藪下 正美
概要

スマホアプリをウェブ技術のみで作れることがFirefox OSの持つ最大の特徴です。モバイル向けのウェブサイト作成に慣れている方であれば、さっとアプリ開発を始められます。Firefox OSの提供するWeb APIを用いることで、従来はできなかったハードウェアの制御、センサーの利用、他のアプリとのデータ共有、ローカルへの大データの保存と管理といったことが可能になります。本セッションでは、ウェブページとしてのコーディングから、実際に利用可能なアプリまでの例を見つつ、Firefox OSの提供するWeb APIの持つ力やアプリ公開までの手順、最低限必要なツールをご紹介します。

HTML5アプリ開発環境最前線 ～ 開発ツールや SDK からクラウドサービスまで ～

登壇者
  • アドビ システムズ 株式会社 クリエイティブ クラウド エバンジェリスト Andy Hall
  • アシアル株式会社田中 正裕
  • Mozilla Japan テクニカルマーケティング 清水 智公
概要

HTML5 アプリは HTML と CSS、JavaScript を使って作成されます。開発に最低限必要なツールはテキストエディターだけですが、さまざまなツールや SDK を組み合わせることで、より高度なアプリを効率よく開発できるようになります。特にマルチプラットフォームにアプリを展開する際には、プラットフォーム間の差異をいかに気にすることなくアプリを開発できるかが鍵となります。本セッションでは HTML5 アプリの開発の効率性を高めるツールや SDK、クラウドサービスをご紹介します。このセッションを通じて、よりアプリの本質に注力できる開発環境について知っていただければ幸いです。

ウェブがモバイルから更に多様なデバイスへと広がるいま、アプリやサービスの提供方法が問われています。Firefox OS ではブラウザーが OS となりオープンなウェブベースの世界を作っていますが、アプリやサービスの提供側としてはどう対応していくべきでしょうか？

サイトとアプリの関係は？本当に HTML5 でアプリを開発できるのか？そもそも Firefox OS には期待して良いのか？国内を代表するサービスやアプリを提供している 3 社を招き、ケータイジャーナリストの石野さんを司会として Firefox OS を語り尽くしていただきます。
