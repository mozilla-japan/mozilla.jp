Firefox Developers Conference 2012 in Osaka — それ、もうウェブでやったらええんちゃう？

開催主旨

Mozilla Japan 主催の Firefox Developers Conference が 1 年ぶりに関西にやってきます！ 昨年 5 月にも大阪で開催したこのイベントは、Mozilla の製品やテクノロジー、ウェブ技術や Open Web に関心の高いデベロッパーやデザイナー、パワーユーザーを対象に、最新のテクノロジーのデモや講演、情報交換を行う場でもあります。今回は、プラットフォームとして急成長を続けるウェブのこれからを知った上で、開発者ツールのあっと驚く使い方やイマドキの HTML の書き方、ウェブプラットフォームでのアプリ配信までじっくり聞いていただける一日です。

カンファレンス終了後には、懇親会も予定していますので、ぜひ奮ってご参加ください。

なお、本編となる本イベントの前日 (6/29) には、CSS Nite in OSAKA も前夜祭として開催されます！

このイベントは終了しました。たくさんのご参加をいただき、ありがとうございました。現在、各セッションの動画と資料を公開しています。

ウェブ開発者ツール、使いこなせてますか？

登壇者
  • 株式会社ジークス テクニカルディレクター 若松 浩昭
  • 株式会社ノイ ウェブディレクター 藤澤 浩子
  • 株式会社パソナテック 福西 藍
概要

ウェブ開発ツールといえば Firefox のアドオン Firebug が知られていますが、機能も多彩でなかなかすべての機能を把握しきれません。また Firebug には強化する拡張機能が多数あり、Firefox 標準の開発ツールも進化を続けています。このセッションでは、ウェブ制作の現場で活躍するディレクターやデザイナーの視点で、開発ツールの実践的な活用法や Tips を質問形式で議論しながら楽しく解説していきます。

才色兼備なグラフィックス SVG が魅せるウェブの未来

登壇者
  • Mozilla Japan ソフトウェアエンジニア Brian Birtles
概要

SVG を使えば、たった一つのファイルで高解像度のタブレットや画面の小さなスマホ上でも美しい画像の表示が可能です。また、画像中の好きなパーツだけを移動・変形することも自由自在ですから SVG でグラフィカルなウェブアプリも簡単に作成できます。今では 最新ブラウザーすべてでサポートされるようになったこの SVG について、実際の事例も参考にしつつ、皆さんと一緒にその活用法や、他の技術との使い分けを学びたいと思います。さらに、SVG と関連技術の標準化動向や実装状況など SVG の未来についてもご紹介します。

ウェブがプラットフォームになるとアプリケーションの開発や配信システムはどのように変わるのでしょうか。アプリを作るにはどうすればいいのか？マルチデバイス対応はどうするのか？これまでのウェブ開発と何が変わるのか？インストール方法はどうなるのか？配信・販売するマーケットはどうなるのか？ウェブ開発とアプリ開発の境界がなくなる時代を前に、開発者の疑問にまとめてお答えします。

Writing HTML ～これからの HTML の書き方～

登壇者
  • 株式会社ミツエーリンクス R&D 本部 取締役本部長 木達 一仁

HTML5 や CSS3 によってウェブデザインの自由度が高まる一方で、モバイルを含むマルチデバイス、アクセシビリティ、セマンティクス、ソーシャルや検索エンジンのメタデータなど、考えることも多くなっています。そんな中で私たちはどのように HTML を書いて行くべきでしょうか？ このセッションでは「モバイルファースト」や「レスポンシブウェブデザイン」などの考え方を踏まえて、これからの HTML の書き方を再検討します。
