Firefox Developers Conference 2006
- Create our future Web browser! -

本イベントは終了いたしました。多数のご参加ありがとうございました。当日使用したプレゼンテーションの PDF ファイルを下記「プログラム」よりご覧いただけます。

Mozilla Japan では、来る 12 月 9 日、「Firefox Developers Conference 2006 - Create our future Web browser! -」と題したイベントを開催いたします。当カンファレンスは、未来のウェブブラウザーの創造に興味・関心のある方々の情報交換の場です。

米国 Mozilla Corporation から Firefox 3 の開発を担当している Stuart Parmenter と Vladimir Vukicevic のエンジニア両名が来日し、2007 年リリース予定の同製品と、それに搭載される新しいレンダリングエンジン Gecko 1.9 についてお話しします。また、日本において Mozilla のコミュニティ活動を運営しているメンバーからの活動報告や、拡張機能作者によるプレゼンテーションも行われます。さらに、Google、価格.com、goo といった、ウェブサービスの分野で活躍する企業の開発担当者をお招きし、各社製品・サービスの最新情報をお話しいただきます。

堅苦しい会議ではなく、あくまでもカジュアルな交流の場を提供したいと考えています。皆様、奮ってご参加ください。
