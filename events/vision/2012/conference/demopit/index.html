DemoPit

Confernce Day 会場のラウンジ内に設けるデモコーナー。「オープンな考えを用いた新しい取り組み」をテーマに、Mozilla 関連技術やウェブテクノロジーのデモに限らず、ハードウェアやアート、アカデミックな取り組みなどについて紹介を行うブースが複数出展します。DemoPit は、午後のセッション時間と懇親会中の見学が可能です。

出展者一覧

Phroni (フローニ)

スマートフォンやタブレットでブラウジング中に検索したいキーワードがあった場合、該当するキーワードをホールド・スワイプで選択し、検索ボックスにコピー＆ペーストし、検索し、リンクを選択するという面倒な動作が必要です。Phroni は、人工知能を使ってウェブページを自動に解析し、キーワードを自動的にリンク化、リンクをタップするだけで様々なサイトに対して検索できるテクノロジーです。現在は Firefox のアドオンで利用可能です。

signup.phroni.com

Paintⁿ (エヌペイント)

Mozilla のキャラクター、Dino やフォクすけとあなたの名前でオリジナルのステンシルテンプレート (型版) を作りましょう！ 翌日のワークショップデーでそれぞれのステンシルを製作し、会場の大きな壁にみんなでペイントして一枚の絵を描きます。

para-site.jp

プロジェクションマッピング協会

ヨーロッパでは盛んに行われるプロジェクションマッピングですが、私たちは、この映像テクニックを若手アーティストに伝授して行きたいと思い、活動を行っています。本来なら，屋外の大型建造物に、夜、オリジナル映像を投影しますが、当協会では、プロジェクションマッピングの可能性を広げるために室内での常設用展示を可能にしました。

www.projection-mapping.jp

CHORDxxCODE (コードコード)

CHORDxxCODE は、東京大学の女性研究者 7 人から構成されるものづくり集団です。開発スタイルとして「おしゃべり」を重視し，そこで得たアイディアを理論化「おさほう」し、実装「おつくり」するオープンな場として“ヒューマンエンジニアリングサロン”という活動を提唱しています。これまで様々な企業や場にお邪魔して「おしゃべり」や「おつくり」を繰り広げてきた CHORDxxCODE のヒューマンエンジニアリングサロンを Mozilla Vision 2012 で特別公開いたします。

chordxxcode.com

Domova (ドムーバ)

ウェブブラウザーの「ホームボタン」、「ブックマーク」、「戻るボタン」などのアイコンや背景の画像を自分で手書きした絵やスタンプでデコっちゃおう！ ぬり絵感覚で、自分だけのオリジナルブラウザーづくりが楽しめるワークショップです。

domova.mozlabs.jp

Mobile Learning Course Designer Street

このプロジェクトは、Moblile Learning 教材の開発者・利用者を結びつけるコミュニティを構築することにより、教材開発のための情報共有や教材活用の促進を支援するためのプロジェクトです。教材に対してユーザーによりタグ付けされた教材のメタデータを用いて、ユーザーの学習進度や興味関心・日常生活におけるコンテクスト等に応じた教材の検索・推薦を実現します。

mldst.phpfogapp.com

Hackasaurus (ハッカザウルス)

「X-Ray ゴーグル」というツールを使って、さまざまなウェブページをまるで魔法のように書き替えたり、イメージを変えたりしながら「ウェブサイトのしくみ」を学ぶことができます。

hackasaurus.org

Popcorn (ポップコーン)

Popcorn では、クリックなどの簡単な操作で、動画とウェブを連携させることが可能になります。たとえば、動画でニュースを見ているときに、その出来事がどこで起きたのかを知りたければ動画のクリックで地図を表示するといったことができるツールです。

mozillapopcorn.org

Scratch (スクラッチ)

これからは「読み・書き・プログラミング」。MIT メディアラボが開発した「スクラッチ」でゲームを作りましょう！ はじめての人でも大丈夫。ワールドビジネスサテライトでも紹介されました！

scratch-ja.org

RunJumpBuild (ランジャンプビルド)

ウェブブラウザーで、誰でも簡単に横スクロールアクションのゲームを作ることができます。好みの画像や音を使い、自由な発想で楽しいオリジナルゲーム作りにチャレンジしてみましょう！

runjumpbuild.com

Lxe² (エルバイイーツー)

Lxe² は、クラウドエンジニアとしてのスキルを診断する IT 検定です。当日はお客様が実際に PC を操作して、IT 検定を体験していただきます。

www.lxe2tm.com
