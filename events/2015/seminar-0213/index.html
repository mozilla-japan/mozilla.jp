アプリ提供事業者向け Firefox OS セミナー

Firefox OS の可能性とアプリ開発のノウハウ

2015 年 2 月 13 日 (金) に、アプリ提供事業者の皆様を主な対象とした Firefox OS に関する無料セミナーを開催いたします。

HTML5 や JavaScript ベースで開発ができ、IoT / WoT との親和性が高い Firefox OS アプリ開発の利点や Firefox OS 市場のご紹介、アプリ配信マーケットである Firefox Marketplace への登録方法などを包括的にご紹介いたします。

概要

ウェブをベースとしたオープンなモバイル OS として、2013 年から各国オペレーターや端末メーカーにより端末提供が始まった Firefox OS。日本でも初の商用端末が 2014 年 12 月に KDDI から発売され、その動向に注目が集まっています。

Mozilla Japan では、来る 2 月 13 日 (金) に、Firefox OS の基本情報からアプリ開発のノウハウをテーマにした無料セミナーを開催いたします。

すでに iOS や Android のアプリを開発されている方はもちろん、ウェブコンテンツサービス事業者の方にも HTML5 と JavaScript でのアプリケーション開発が可能になる Firefox OS の世界をご紹介できればと思います。

本セミナーでは、Firefox OS の概要からアプリ作成手順、マーケットプレイスへの公開方法などついてのご紹介を行う予定です。

なお、当日は、会場に Firefox OS の開発リファレンス端末「Flame」をご用意しておりますので、購入ご希望の方には休憩時間に販売させていただきます。

Firefox OS 概要

登壇者
Mozilla Japan
マーケティングマネージャー
安江 秀介
概要

Firefox OS は、2013 年の MWC で KDDI などのパートナー企業拡大を発表以来、世界中で主にスマートフォン向けに活用されてまいりました。
日本でも、昨年末 KDDI 様から Firefox OS 搭載スマートフォン Fx0 がリリースされ、さらに 2015 年1 月の CES ではパナソニック様から Firefox OS 搭載 TV リリースのアナウンスがありました。
このように益々広がりを見せる Firefox OS ですが、この OS 自体は HTML5、CSS、JavaScript といった一般的なウェブ技術をベースに開発された OS です。
また Firefox OS 上で動作するアプリも同様にウェブ技術がベースとなります。
開発にかかる負担が少ないだけではなく、Firefox OS のみならず様々なプラットフォーム上で動作可能であるため、現在の IoT(Internet of Things) の動向からも、将来的に大きな可能性を秘めております。
今回、私たち Mozilla の Firefox OS への取り組みや取り巻く動向、そして Firefox OS に向けたアプリ開発の魅力と可能性についてお伝えさせていただきます。

Firefox OS アプリ開発方法について

登壇者
Mozilla Japan
ソフトウエアエンジニア
吉永 満太朗
概要

Firefox OS アプリは HTML5 / JavaScript / CSS などの一般的なウェブ技術で構築できます。
すでにウェブアプリをお持ちの場合は、Manifest ファイルを追加するだけで Firefox OS アプリとして公開でき、 開発環境の構築や開発のための知識習得が容易にできることが特徴です。
本セッションでは、Firefox OS アプリ開発の概要や、一般的な Android / iOS アプリとの相違点、ウェブアプリを Firefox OS アプリとして公開するための方法等をご紹介します。
また、アプリ開発をする上での注意点や、開発事例などをご紹介し、すぐに Firefox OS アプリを始めるための包括的な情報をご説明します。

Firefox Marketplace におけるアプリの公開と管理について

登壇者
Mozilla Japan
テクニカルマーケティング
清水 智公
概要

Firefox OS 向けのアプリを配信するプラットフォームが Firefox Marketplace です。
Firefox OS 端末には Marketplace アプリがプリインストールされており、ユーザーはそこからアプリを検索し、自分の端末へインストールします。
このセッションでは作成した Firefox OS アプリの Firefox Marketplace への公開と、そこでのアプリの管理についてご説明します。
また Marketpalce でのレビューの方針や、申請時における注意点などもあわせてご説明いたします。
