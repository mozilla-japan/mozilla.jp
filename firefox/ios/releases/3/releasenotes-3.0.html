iOS 版 Firefox リリースノート

バージョン — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。全変更点はこちらでご覧になれます。

このバージョンでの変更点

  • 新機能

    Apple Map や Twitter のようなアプリを起動するリンクの処理が向上しました

  • 新機能

    新規タブ画面に標準で表示されるタイルを削除できるようになりました

  • 新機能

    新規ユーザーには Alexa のトップ 5　サイトが新規タブ画面に表示されるようになりました

  • 新機能

    インラインへの入力時の自動補完対象に、標準のドメインが含まれるようになりました

  • 新機能

    パスワードマネージャーが、Touch ID / パスコードに対応しました

  • 新機能

    オランダ語 [da] がロケールに追加されました
