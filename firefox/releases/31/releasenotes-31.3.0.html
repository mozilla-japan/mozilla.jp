Firefox リリースノート

バージョン 延長サポート版 (ESR) — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンでの変更点

  • 修正

    いくつかのセキュリティ問題が修正されました。

  • 修正

    @ JS::Handle::operator JSObject* const&() が起動時にクラッシュする不具合を修正しました。 (Bug 1055766)

  • 修正

    アドオンマネージャーが断続的に mochitest (ブラウザーテストの一種) に失敗する不具合を修正しました。 (Bug 1095128)

  • 修正

    Yosemite 上で mochitest を実行すると、Bad CPU type in executable と出力されテストに失敗する不具合を修正しました。(Bug 1054043)

  • 修正

    Mac OS 10.10 で nsChildView.mm のビルドに失敗する不具合を修正しました。 (Bug 1005458)

  • 修正

    いくつかの x86 CPU で CPU の機能を誤って検出することがある不具合を修正しました。(Bug 1096651)

  • 修正

    Yosemite 上のビルドエラーを修正しました。 (Bug 1045231)

  • 修正

    XMLHttpRequest.send({}) が送信されてしまう不具合を修正しました。 (Bug 1096263)
