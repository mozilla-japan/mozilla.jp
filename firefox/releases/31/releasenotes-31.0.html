Firefox リリースノート

バージョン — リリース

Firefox に貢献いただいたすべての方に感謝いたします。新たに貢献された方々はこちらからご覧になれます。

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    新しく開いたタブに検索欄が追加されました。

  • 新機能

    mozilla::pkix が標準の証明書検証器となりました。(詳細はこちら)

  • 新機能

    ダウンロードされたファイル中のマルウェアがブロックされるようになりました。(詳細はこちら)

  • 新機能

    OpenType MATH table の一部分が実装されました。詳細は数学フォント、および MathML Torture テスト に関する文書をご覧ください。

  • 新機能

    Windows: アプリケーションが設定されていない場合、Firefox が ogg ファイルがの再生および pdf ファイルの表示を行います。

  • 変更

    capability.policy.* を利用してサイト特有の許可設定を行う CAPS が削除されました。最も大きな注意点は、CAPS を利用してクリップボードへのアクセス許可を行えなくなりました。checkloaduri は唯一の例外として、以前と同様、file:// URL からのロードの許可のために利用できます。

  • 変更

    日本語ロケール：検索エンジンに Bing が追加されました。

  • HTML5

    WebVTT が実装され、利用できるようになりました。 (詳細はこちら)

  • HTML5

    CSS variables が実装されました。(詳細はこちら)

  • 開発者

    開発ツール：アドオンデバッガーが利用できるようになりました。(詳細はこちら)

  • 開発者

    開発ツール：Canvas デバッガーが利用できるようになりました。(詳細はこちら)

  • 開発者

    Array.prototype.fill() が利用できるようになりました。 (詳細はこちら)

  • 開発者

    Object.setPrototypeOf() が利用できるようになりました。(詳細はこちら)

  • 開発者

    CSP 1.1 の nonce-source と hash-source が標準で利用できるようになりました

  • 開発者

    開発ツール：カラーピッカーにスポイトが追加されました。(詳細はこちら)

  • 開発者

    開発ツール：表示されているボックスモデル中の数値を編集できるようになりました。(詳細はこちら)

  • 開発者

    開発ツール：エディターにさまざまな機能が追加されました。(詳細はこちら)

  • 開発者

    開発ツール：コンソールにスタックとレースが表示できるようになりました。(詳細はこちら)

  • 開発者

    開発ツール：cURL 形式でコピーできるようになりました。(詳細はこちら)

  • 開発者

    開発ツール：コンソールへログをスタイル付きで出力できるようになりました。(詳細はこちら)

  • 開発者

    navigator.sendBeacon が標準で利用できるようになりました。(詳細はこちら)

  • 開発者

    onbeforeunload イベントで生成されたダイアログが、ブラウザーの他の部分へのアクセスをブロックしなくなりました。

  • 修正

    部分的に選択されたリンクテキストを利用したコンテキストメニューからの検索に関する不具合を修正しました。(Bug 985824)

既知の問題

  • 未解決

    Firefox の終了が遅いため、「Firefox は起動していますが応答しません」と表示される場合があります。 (Bug 966469 および Bug 985655)

  • 未解決

    PDF.js：いくつかのフォントで、いくつかの文字が表示されないことがあります。この現象はごく少数の PDF ファイルで発生します。 (Bug 1028735)

  • 未解決

    Mac OS および Windows：Citrix Receiver が動作しません。回避策として、アドオンマネージャーでプラグインを常に有効にしてください。(Bug 1025627)

  • 未解決

    GNU/Linux および Windows XP：Google Maps のストリートビューが正しく動作せず、画面が黒くなることがあります。 (Bug 1034593)

    • Firefox 32.0a2で解決
  • 未解決

    Mac OS X: ウィンドウがない場合に Cmd-L を押しても新しいウィンドウが開かれません。 (Bug 1008793)

    • Firefox 32.0a2で解決
