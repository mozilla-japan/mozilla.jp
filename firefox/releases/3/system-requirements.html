Firefox システム要件

Firefox 3

Windows

オペレーティングシステム (OS)

  • Windows 2000
  • Windows XP
  • Windows Server 2003
  • Windows Vista

最小ハードウェア構成

  • Pentium 233 MHz プロセッサー (Pentium 500 MHz 以上を推奨)
  • 64 MB の RAM (128 MB 以上を推奨)
  • 52 MB のハードディスク空き容量

Mac

オペレーティングシステム (OS)

  • Mac OS X v10.4 以上

最小ハードウェア構成

  • Intel x86 または PowerPC G3/G4/G5 プロセッサーを搭載した Macintosh コンピューター
  • 128 MB の RAM (256 MB 以上を推奨)
  • 200 MB のハードディスク空き容量

Linux

ソフトウェア要件

注: 各 Linux ディストリビューターは、それぞれのディストリビューションに対して、システム要件が異なるパッケージを提供している場合もあります。

  • Linux カーネル 2.2.14 と、最低限以下のライブラリまたはパッケージが必要です。
    • glibc 2.3.2 以上
    • XFree86-3.3.6 以上
    • gtk+2.0 以上
    • fontconfig (別名 xft)
    • libstdc++5

最小ハードウェア構成

  • Pentium Pentium II もしくは AMD K6-III+ 233 MHz CPU (500 MHz 以上を推奨)
  • 64 MB の RAM (128 MB 以上を推奨)
  • 52 MB のハードディスク空き容量
