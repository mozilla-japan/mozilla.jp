Firebird 0.7.1 for Mac OS X

Mozilla Firebird 0.7.1 は、Mozilla Firebird 0.7 の Mac OS X 向けサービスアップデートです。その他のプラットフォームをお使いの方は Mozilla Firebird 0.7 をご利用ください。

Mozilla Firebird 0.7.1 for Mac OS X はテクノロジープレビューです。このソフトは、ほとんどの場合正常に機能し、デフォルトブラウザーとして信頼できるほど十分なレベルにありますが、バージョンが 1.0 以前の段階であり、パフォーマンスや安定性については一切保証できませんので、基幹システムとして信頼すべきではありません。詳しくは ライセンス規約 をご覧ください。

このドキュメントには、Mozilla Firebird 0.7.1 のリリースに関する 新機能、ダウンロード・インストール方法、既知の問題、よくある質問 が載っています。Bugzilla にバグを報告する前に、このリリースノートと バグ登録方法 をお読みください。

Mozilla Firebird に関するあなたのフィードバックをお待ちしています。MozillaZine [日本語版] が親切にも提供してくれている Mozilla Firebird フォーラム にぜひ参加してください。

新機能

今回のサービスリリースでは、0.7 に含まれていた以下のバグを修正しました。

  • デフォルトブラウザーとして設定した場合に、他のアプリケーションから送られた URL が開けない (XML エラーウィンドウが表示される)
  • Save Page As、Save Image As、Save Link As の各メニューが機能しない
  • Pinstripe テーマ使用時に、タブをダブルクリックすると新しいタブが開かれる
  • デスクトップへのファイルの自動ダウンロードがデフォルトで無効になっている
  • すべてのウィンドウを閉じたあと、最初に開いた新しいウィンドウで、ユーザーインターフェイス (ブックマーク、ナビゲーション、セッション履歴など) が正しく機能しない

今回のリリースではパフォーマンスの改良もいくつか行われ、Mac OS X 10.2 "Jaguar" と 10.3 "Panther" では、システムの速度とメモリーによっては 15% から 40% も起動時間が短くなりました。

ダウンロード・インストール・アンインストール

Mac OS X: firebird-0.7.1-mac.dmg.gz

ファイルをダウンロードしてディスクイメージをマウントし、アプリケーションファイルを置きたい場所にドラッグしてください。

0.8 マイルストーンに向けて、Mac OS X 版のデフォルトテーマを Kevin Gerich の Pinstripe テーマに移行する予定です。それまでの間、Aqua インターフェイスを引き立たせるために、Mozilla Firebird 0.7.1 に Pinstripe Beta をインストールすることを強くお勧めします。

Mozilla Firebird をシステムから削除するには、アイコンをゴミ箱にドラッグしてください。

既知の問題

ここには、Mozilla Firebird 0.7.1 に関する既知の問題の一部が載っています。新たにバグを報告する前にこのリストをお読みになり、今後も定期的にチェックするようにしてください (リリースに新しいバグが見つかった場合は更新されます)。

  • Mozilla Firebird は Mozilla Application Suite から多くのコードを利用しています。Mozilla Firebird に含まれる問題の多くは、実際には Mozilla のコアコードにおける問題です。ページのコンテンツや接続性に関する問題を発見した場合、それはおそらく Mozilla の問題ですから、Bugzilla では Mozilla Firebird プロダクトではなく Browser プロダクトに対して報告してください。
  • テーマインストールの "Use this theme" チェックボックスは正しく機能しませんので、Tools > Options > Themes から切り替える必要があります。また、一部のテーマは再起動しないと完全に切り替わらないことがあります。
  • フォームのオートコンプリート (自動補完) を無効化するオプションは存在しますが、ユーザーインターフェイスを無効にするだけで、フォームに入力したデータは収集されてしまいます。今後のリリースで、データの収集も完全に無効になる予定です。
  • Mac OS X 版の Mozilla Firebird は新しく、まだ完成度が十分ではありません。一部のユーザーインターフェイスは Windows のようなままです。Mozilla Firebird チームは Mac OS X の熱烈な支持者であり、Mac OS X においてもできる限り最高のユーザー経験を提供したいと考えています。バージョン 1.0 までに、このための最適化と機能強化を行う予定です。

Mozilla Firebird に関するその他の問題、よくある質問、便利な使い方、一般的なヘルプについては、David Tenser の大変便利なヘルプサイト Mozilla Firebird [日本語版] や、MozillaZine [日本語版] によって運営されている Mozilla Firebird フォーラム をチェックしてください。

よくある質問

  1. Mozilla Firebird が Mac OS X では質の悪い移���に見えるのはなぜですか?

    Mozilla Firebird はこれまで、Mac OS X 上での開発について、それほど強い関心を浴びてきませんでした。これは、開発チームが Macintosh コンピューターを持っていなかったため、あるいは他の作業で忙しかったためです。最近になってこの状況は一変しました。私たちが現在利用しているコードは Aqua ユーザーインターフェイスガイドラインに最適化されていませんが、特に新世代の Apple ハードウェアにおいては大きな将来性を持っています。Mac OS X でのユーザー経験を最適化する作業は今後数か月の間に実施されます。それまでの間、Mac 用の Pinstripe テーマ をインストールして、ユーザーインターフェイスの大部分のルック＆フィールを向上させることができます。

  2. どのような協力ができますか?

    私たちは少しでも多くの配布経路を必要としています。5 人の友人や家族、同僚を Mozilla Firebird に移行させることがあなたの使命です。もしあなたが学生なら、学校で Mozilla Firebird を配布してください。リリースについて Slashdot [日本語版] やその他のニュースサイトに記事を投稿してください。あなたの blog から情報を流してください。例えばあの Kazaa のように、インターネットを通じた大規模な配布が可能です。Firebird の話題をどんどん広めてください!

  3. 私は別のブラウザーを使っています。なぜなら...

    もしあなたが Mozilla Firebird を使っていないのなら、開発チームにその理由を教えてください。私たちは Mozilla Firebird フォーラム に寄せられたフィードバックに目を通しています。

  4. なぜ私が送ったメールに返事をくれないのですか?

    フォーラム をご利用ください。Firebird チームは、フォーラムに寄せられたフィードバックを欠かさず読んでいます。メールを送ってもおそらく無視されます。

  5. テーマとアドオン (拡張機能) はどこで手に入りますか?

    テーマ と 拡張機能 をご利用ください。

  6. Mozilla Firebird の作業をしているのは誰ですか?

    Dave Hyatt、Blake Ross、Pierre Chanial、noririty、Ben Goodger、Dean Tessman、Mike Kaply、Jan Varga、Brian Ryner が開発に参加しています。Asa Dotzler は、リリース、バグの選別、ウェブサイト、その他雑用を手伝ってくれます。David Tenser はドキュメントやウェブサイト、リリースに関して協力してくれます。Arvid Axelsson はデフォルトテーマをメンテナンスしています。Mozilla Firebird に関する評価、協力の申し出、その他一般のメールは フォーラム に送ってください。

  7. Mozilla Firebird のソースはどこにありますか?

    cvs.mozilla.org にあります。Mozilla トランクと mozilla/browser と mozilla/toolkit を組み合わせたものになります。

  8. Mozilla Firebird は、ユーザーインターフェイスにちょっとだけ手を加えた、ただの Mozilla では?

    すでに 4 万行以上のコードが Mozilla のコードに追加・変更されていますので、別物として捉えるべきです。

  9. Phoenix の名前に何が起きたのですか?

    商標の問題から名称を変更する必要がありました。新名称は Mozilla Firebird です。

  10. メールクライアントはどこにあるのですか?

    現在フルセットの Mozilla アプリケーションスイートを使っている人の多くは、それをメールやニュースの購読にも利用していると思います。Mozilla Firebird にはブラウザー機能しかありませんので、そのような利用はできませんが、Mozilla Firebird への乗り換えたからといって Mozilla のメールクライアントの利用をあきらめる必要はありません。Mozilla Firebird ブラウザーにぴったりな、新しいメールアプリケーション Mozilla Thunderbird をご利用ください。
