このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    アドオンの互換性確認をより柔軟にしたことで、ほとんどのアドオンが新バージョンでも引き続き利用可能となりました。

  • 新機能

    履歴を戻るまで [進む] ボタンを表示しないようにしました。

  • 新機能

    Mac で ATOK の確定アンドゥが使用できるようになりました。ただし、キーボードショートカットが変更されていると動作しない場合があります。(Bug 477291)

  • 新機能

    WebGL のアンチエイリアスを実装しました。(Bug 615976)

  • 新機能

    CSS3 の 3D トランスフォームを実装しました。(Bug 505115)

  • HTML5

    双方向テキストの分離を実現する HTML5 新要素 bdi を、CSS プロパティとともに実装しました。(Bug 613149、Bug 662288)

  • HTML5

    全画面での実行に対応したウェブアプリケーションを開発できる フルスクリーン API を実装しました。

  • 開発者

    仕様との整合性を高めた IndexedDB API を実装しました。

  • 開発者

    ウェブ開発者向け新機能として、該当要素の強調表示や CSS スタイルの確認を行える 調査ツール を追加しました。

  • 修正

    Mac でブックマークを移動中にクラッシュする場合がある問題を修正しました。(Bug 681795)

  • 修正

    Mac で、Apple から公開された最新の Java をインストールした後、Java アプレットが使われているページのタブを閉じたときにクラッシュする場合がある問題を修正しました。(Bug 700835)

  • 修正

    一部の Mac ハードウェアで Silverlight 動画が再生されない問題を修正しました。(Bug 715396)

  • 修正

    いくつかの安定性に関わる問題を修正しました。

  • 修正

    いくつかのセキュリティ問題 を修正しました。

既知の問題

  • 未解決

    ブラウザーのバージョン番号が 2 桁になったことで、一部のサイトではバージョン判定に失敗する サイト互換性の問題 が生じる可能性があります。(Bug 690287)

  • 未解決

    ロックされたプロファイルで Firefox を起動しようとするとクラッシュします。(Bug 573369)

  • 未解決

    一部のユーザーから、Gmail メイン画面上のでのスクロールが通常より遅くなるとの報告があります。(Bug 579260)

  • 未解決

    一部の Synaptics 社製タッチパッドで垂直スクロールができません。(Bug 622410)

  • 未解決

    Growl 1.3 以降で Firefox の通知が表示されない場合があります。(Bug 691662)

    • Firefox 10 では未解決
    • Firefox 11 で解決
  • 未解決

    特定の状況で、スクロールやテキスト入力がぎくしゃくした状態になることがあります。(Bug 711900)

  • 未解決

    Java アプレット上で文字入力が無反応になる場合があります。(Bug 718939)

    • Firefox 10 では未解決
    • Firefox 10.0.1 で解決
