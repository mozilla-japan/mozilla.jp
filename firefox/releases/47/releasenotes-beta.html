バージョン ベータ版 — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    Flash がインストールされていない場合、ページに埋め込まれた Youtube の動画再生を HTML5 の video 要素を用いて行うようになりました

  • 新機能

    ChaCha20/Poly1305 暗号を利用するサイトに対応しました

  • 新機能

    Windows 版と Mac 版で Google の Widevine CDM を利用できるようになりました

  • 新機能

    高速なマシンをご利用の場合、VP9 コーデックを利用できるようになりました

  • 新機能

    登録されている ServiceWorker を起動するボタンが、ServiceWorker 向けの開発ツールにつきました

  • 新機能

    about:debugging#workers で登録済みの ServieWorker の一覧とデバッグができるようになりました

  • 新機能

    プッシュメッセージをシミュレートする機能が about:debugging に実装されました。テストにご利用になれます

  • 新機能

    コンソールで複数行の入力が可能になりました

  • 新機能

    about:performance で Firefox のパフォーマンスを確認できるようになりました

  • 新機能

    WebCrypto：RSA-PSS 署名に対応しました

  • 新機能

    進む、戻るを行った際に、https で配信されるリソースの no-cache が可能になりました

  • 変更

    FUEL (Firefox User Extension Library) が削除されました。これにより依存するアドオンが動作しなくなります。add-on SDKをご利用ください

  • 変更

    e10s の性能に関する問題を避けるため、browser.sessionstore.restore_on_demand preference の値が標準の true にリセットされます

  • 変更

    WebCrypto：PBKDF2 が SHA2 アルゴリズムをサポートしました

  • 開発者

    Tilt が AMO で公開されました

  • 開発者

    変更によるアドオン互換性に関する情報はこちらをご覧ください

  • 開発者

    TextTrack オブジェクトで cuechange イベントが利用できるようになりました

  • 開発者

    レスポンシブデザインモードでユーザーエージェントを変更できるようになりました
