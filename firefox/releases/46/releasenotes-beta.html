このバージョンでの変更点

  • 新機能

    HTTP で配信されているログインフォーム付きのサイトは、「安全でない接続」と表示されるようになりました

  • 新機能

    JavaScript の Just In Time (JIT) コンパイラがセキュリティの面で改良されました

  • 新機能

    GNU/Linux: GTK3 が統合されました

  • 変更

    WebRTC の性能と安定性が向上しました

  • 開発者

    メモリーツールで、ドミネーターツリーを表示するようになりました

  • 開発者

    スタイルエディターの @media ルールサイドバーから、レスポンシブデザインビューを起動できるようになりました

  • 開発者

    パフォーマンスパネルでメモリー割り当てやガーベジコレクションを確認できるようになりました

  • HTML5

    documents.elementsFromPoint に対応しました

  • HTML5

    Web Crypto API が HKDF へ対応しました
