Firefox リリースノート

バージョン — リリース

Firefox に貢献いただいたすべての方に感謝いたします。新たに貢献された方々はこちらからご覧になれます。

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    証明書がエラー起きた場合、もしくは接続の安全性を確認できない場合に表示される警告画面が改善されました

  • 新機能

    デコーダがインストールされていれば、Linux で H.264 が利用できるようになりました

  • 新機能

    MP4/H.264 が利用できない場合、WebM/VP9 が有効になります

  • 新機能

    コンポジットスレッドで動作するアニメーションは、アニメーションインスペクターのタイムラインにイナズマアイコン付きで表示されるようになりました

  • 新機能

    HTTPS でのコンテントエンコーディングで、brotli 圧縮が利用できるようになりました

  • 新機能

    開発ツールのスクリーンショットコマンドで、ピクセル比を選択できるようになりました

  • 修正

    Windows XP と Vista で、動画再生時にスクリーンセーバを無効にできない不具合が修正されました (Bug 1193610)

  • 修正

    いくつかのセキュリティ問題 を修正しました。

  • 変更

    ウェブフォントの Unicode レンジに対応するため、Linux におけるフォントマッチングのコードが他のプラットフォームと共通化されました

  • 変更

    新しい要件に適合するため、Windows 版で SHA-256 で署名された証明書を利用するようになりました

  • 変更

    RC4 復号器をサポートしなくなりました

  • 変更

    サイト証明書の検証時に、Equifax Secure Certificate Authority の 1024 ビットルート証明書、および UTN - DATACorp SGC を信頼できる証明書としなくなりました

  • 変更

    ウェブフォントの検証をより厳密に行うようになりました

  • 変更

    Windows 8、および Windows 8.1 でスクリーンキーボードが利用できなくなりました。 これは一時的な措置です。

  • 開発者

    ログに出力されたオブジェクトで右クリックすると、ページ内のグローバル変数として参照できるようになりました

  • 開発者

    アニメーション用のツールが追加されました：

    • インスペクターで CSS アニメーションのキーフレームを表示、編集できるようになりました
    • アニメーションの動きを決める 3 次ベジェ関数を、視覚的に編集できるようになりました
    • CSS アニメーションとトランジションの、再生および一時停止、シークができるようになりました
    • 詳しくは http://devtoolschallenger.com/ をご覧ください
  • 開発者

    レイアウトとスタイルを視覚的に調査するツールが実装されました：

    • 物差しをビューポートの際に表示することが可能になりました。またスペースや並びの問題を発見を容易にする測定ツールが追加されました
    • ドロップシャドウや、セピアといった CSS フィルターを、リアルタイムで作成とプレビューができるようになりました
    • 詳しくは http://devtoolschallenger.com/ をご覧ください
  • 開発者

    ヒープの使用状況を調査できる新しいメモリーツールが実装されました

  • 開発者

    Service Workers APIが利用できるようになりました

  • 開発者

    JSON リーダが組み込まれ、拡張なしでデータの閲覧、検索、コピー、そして保存ができるようになりました

  • 開発者

    デバッガーで Cmd キーを押しながらクリックすると、その関数の定義にジャンプするようになりました

  • 開発者

    WebSocket Debugging API が利用できるようになりました。これを利用したアドオンの作例も公開されました

  • 開発者

    ルールビューで編集されたルールがスタイルエディターにも表示されるようになりました。また編集されたルールがルールビューに表示されるようになりました
