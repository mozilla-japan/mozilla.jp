バージョン ベータ版 — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    リーダービューがさらに読みやすくなりました

  • 新機能

    ハードウェアアクセラレーションのない OS X での性能が向上しました

  • 新機能

    Firefox Hello が廃止されます

  • 新機能

    OS X での文字のアンチエイリアス表示が改善されました

  • 新機能

    Graphite2 が標準で有効になりました

  • 新機能

    ハードウェアアクセラレーションがないシステムでも SSSE3 命令セットがサポートされていれば、動画のパフォーマンスが向上します

  • 新機能

    OS X：新規にインストールした場合、管理者権限を持つ場合にのみ Firefox 更新できるようになりました

  • 新機能

    リーダービューでの幅と、行の高さを調整しました

  • 新機能

    保存された HTTP でのログイン情報が、HTTPS でアクセスした場合でも提示されるようになりました

  • 新機能

    HTML5 の音声と動画に関する改善が行われました

    • 組み込みのコントローラ上で利用できるコンテキストメニューから、ループ再生ができるようになりました
    • コンテキストメニューから 1.25 倍速で、音声と動画を再生できるようになりました
  • 変更

    OS X 10.6、10.7、10.8 のサポートが終了しました

  • 変更

    Windows： SSE2 命令セットをサポートした CPU が必須となりました

  • 開発者

    ネットワークモニターに cause が追加されました。ここには、ネットワークリクエストが発生した理由が表示されます

  • 開発者

    Web Speech の音声合成 API が利用できるようになりました

  • HTML5

    イメージマスクとして、SVG の mask 要素を複数利用できるようになりました
