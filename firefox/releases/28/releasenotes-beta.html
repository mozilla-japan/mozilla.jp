Firefox リリースノート

バージョン ベータ版 — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    Windows 8 の動作するタッチデバイス向けユーザーインタフェースを試験的に搭載しました。詳しくはブログをご覧ください。

  • 新機能

    Gamepad API が利用できるようになりました。詳しくはこちらをご覧ください。

  • 新機能

    HTML5 の video 要素や audio 要素で音量を調節できるようになりました。

  • 新機能

    VP9 でエンコードされた動画を再生できるようになりました。

  • 新機能

    Opus を利用した WebM 動画を再生できるようになりました。

  • 新機能

    Mac OS X: 通知センターが web notifications をサポートするようになりました。

  • 変更

    spdy/2 を利用できなくなりました。

  • 変更

    利用者のプライバシー保護のため、navigator.plugins を利用してプラグインを列挙できなくなりました。

  • 開発者

    multi-line flexbox が利用できるようになりました。

  • 開発者

    CSS の position 属性に sticky と実験的に指定できるようになりました。(Auroraのみ) デモはこちらでご覧になれます。

  • 開発者

    MathML 2.0 の mathvariant 属性が利用できるようになりました。

  • 開発者

    バックグラウンドスレッドのハングアップが通知されるようになりました。

  • 開発者

    <input type="color"> を利用できるようになりました。

  • 開発者

    <input type="number"> を利用できるようになりました。

既知の問題

  • 未解決

    Flash ビデオを全画面で再生中にバックグラウンドへ切り替えられた Firefox は再表示時にフリーズします。(Bug 809055)

  • 未解決

    KB2670838 (MSIE 10 を動作させるために必要なアップデート) がインストールされた Windows 7、もしくは Windows 8.1 でのテキスト描画に不具合が生じます。Windows 8.1 での回避策はこちらをご覧ください。(Bug 812695)
