バージョン ベータ版 — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    スマートフォンを貸した時に利用してもらうための、ゲストブラウジング機能が追加されました。

  • 新機能

    混合コンテンツをブロックするようになりました。これにより中間者攻撃や HTTPS 通信を利用するページの盗聴を防げます。

  • 新機能

    アドオンがアドレスバーにアイコンを表示できるようになりました。

  • 変更

    ルーマニア、アイルランド、ウクライナの各ロケールが追加されました。

  • 開発者

    デスクトップ版 Firefox からのリモートデバッグを有効にする項目が設定に追加されました。

  • 開発者

    Contacts API が実装されました。

  • 開発者

    CSS3 の background-attachment:local が実装されました。

  • 開発者

    ECMAScript 6 の関数が多く実装されました。

  • HTML5

    iframe の srcdoc 属性に対する処理が実装され、インラインフレーム内に埋め込めるようになりました。

既知の問題

  • 未解決

    Android 4.3 の動作する Nexus10 では動作しません。(Bug 900020)

  • 未解決

    Flash ビデオを全画面で再生中にバックグラウンドへ切り替えられた Firefox は再表示時にフリーズします。(Bug 809055)

  • 未解決

    Google で最近行われたデザイン変更によって、Firefox など WebKit 以外のブラウザーで問題が生じているところがあります。(Bug 668288)

  • 未解決

    文字サイズの自動拡大機能によって、一部のページで文字サイズが不釣り合いになる場合があります。(Bug 707195)

  • 未解決

    複数のチャンネル (Aurora、Beta など) の Firefox がひとつの端末にインストールされていると、Firefox Sync が期待通りに動作しません。(Bug 761206)

  • 未解決

    いくつかのHTC Gingerbread (Android 2.3) 端末で動画が再生できないことがあります。(Bug 829454)

  • 未解決

    HTC Sense UI (Froyo) 上でのプライベートブラウジング時に、URL バー中の文字が表示されないことがあります。(Bug 833351)
