Android 版 Firefox リリースノート

バージョン ベータ版 — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    セーフブラウジングを有効化した最初のモバイルブラウザーになります。

  • 新機能

    Google Now 検索ウィジェットを統合しました。

  • 新機能

    Updater を改善して、Firefox 更新時の動作を改善しました。

  • 変更

    Firefox に新しいフォントを同梱して読み込ませることができるようになりました。

  • 開発者

    CSS3 Flexbox を実装しました。

  • 開発者

    新しい DOM プロパティ window.devicePixelRatio をサポートしました。

  • 開発者

    CSS の @supports 規則をサポートしました

  • 修正

    HTTPS ページで安全でないコンテンツの読み込みを無効化しました。(Bug 62178)

  • 修正

    スマートバーへのテキスト入力時に検索サジェスト有効化のオプトインが実装されました。(Bug 586885)

  • 修正

    Android 4.2: ページ読み込み完了時ではなく、読み込み完了後に画面に触れたときに、TalkBack がページ読み込み完了を知らせる問題を修正しました。(Bug 812480)

既知の問題

  • 未解決

    Google で最近行われたデザイン変更によって、Firefox など WebKit 以外のブラウザーで問題が生じているところがあります。(Bug 668288)

  • 未解決

    Android 版 Firefox はシステムの言語設定と同じロケールを使用します。(Bug 695565)

  • 未解決

    文字サイズの自動拡大機能によって、一部のページで文字サイズが不釣り合いになる場合があります。(Bug 707195)

  • 未解決

    端末の向きを変えたときに、意図しない挙動になる場合があります。(Bug 740691)

  • 未解決

    複数のチャンネル (Aurora、Beta など) の Firefox がひとつの端末にインストールされていると、Firefox Sync が期待通りに動作しません。(Bug 761206)
