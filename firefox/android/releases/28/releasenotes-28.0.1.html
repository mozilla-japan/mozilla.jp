Android 版 Firefox リリースノート

バージョン — リリース

Firefox に貢献いただいたすべての方に感謝いたします。新たに貢献された方々はこちらからご覧になれます。

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 修正

    28.0.1 いくつかの Galaxy 端末で H.264 エンコードされた動画が再生できなかった問題を修正しました。 (Bug 963621)

  • 修正

    28.0.1 file プロトコルのリンクをクリック時の標準動作が SD カードへのダウンロードであった不具合を修正しました。これは Android 版のみの不具合でした。

  • 新機能

    文字の選択、カット、コピーの動作が Android ネイティブのものと同じになりました。

  • 新機能

    ロケーションバー に入力した URL をもとに予測を行うようになりました。

  • 新機能

    複数の共有ボタンが追加されました。

  • 新機能

    OpenSearch が利用できるようになりました。

  • 新機能

    エストニア語 [et] が追加されました。

  • 新機能

    VP9 でエンコードされた動画を再生できるようになりました。

  • 新機能

    HTML5 の video 要素や audio 要素で音量を調節できるようになりました。

  • 新機能

    Opus を利用した WebM 動画を再生できるようになりました。

  • 新機能

    タイトルバーを自動的に隠すかどうかを設定する項目が設定画面に追加されました。

  • 変更

    spdy/3 が実装されました。これは spdy/2 を利用できなくなったためです。spdy/3 に対応しないサーバーとは http/1 で通信を行います。

  • 開発者

    MathML 2.0 の mathvariant 属性が利用できるようになりました。

  • 開発者

    multi-line flexbox が利用できるようになりました。

  • 開発者

    バックグラウンドスレッドのハングアップが通知されるようになりました。

  • 修正

    Explore By Touch を利用時に、ウェブページがロードされてしまう不具合を修正しました。 (Bug 963000)

  • 修正

    いくつかのセキュリティ問題が修正されました。

既知の問題

  • 未解決

    いくつかの Android 4.3 もしくは 4.4 端末上で埋め込まれた h.264 の再生に不具合が生じることがあります。 (Bug 984230)

    • Firefox 28 では未解決
    • Firefox 29で解決
  • 未解決

    Galaxy Tab 3 10.1 (x86 タブレット) 上で、戻るボタンを押すとクラッシュすることがあります。 (Bug 983211)

  • 未解決

    Flash ビデオを全画面で再生中にバックグラウンドへ切り替えられた Firefox は再表示時にフリーズします。(Bug 809055)
