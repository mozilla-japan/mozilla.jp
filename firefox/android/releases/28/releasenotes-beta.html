Android 版 Firefox リリースノート

バージョン ベータ版 — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    文字の選択、カット、コピーの動作が Android ネイティブのものと同じになりました。

  • 新機能

    ロケーションバー に入力した URL をもとに予測を行うようになりました。

  • 新機能

    複数の共有ボタンが追加されました。

  • 新機能

    エストニア語 [et] が追加されました。

  • 新機能

    OpenSearch が利用できるようになりました。

  • 新機能

    HTML5 の video 要素や audio 要素で音量を調節できるようになりました。

  • 新機能

    タイトルバーを自動的に隠すかどうかを設定する項目が設定画面に追加されました。

  • 新機能

    VP9 でエンコードされた動画を再生できるようになりました。

  • 新機能

    Opus を利用した WebM 動画を再生できるようになりました。

  • 変更

    spdy/2 を利用できなくなりました。

  • 変更

    利用者のプライバシー保護のため、navigator.plugins を利用してプラグインを列挙できなくなりました。

  • 開発者

    <input type="color"> と記述することでカラーピッカーを利用できるようになりました。

  • 開発者

    multi-line flexbox が利用できるようになりました。

  • 開発者

    MathML 2.0 の mathvariant 属性が利用できるようになりました。

  • 開発者

    バックグラウンドスレッドのハングアップが通知されるようになりました。

  • 開発者

    <input type="number"> を利用できるようになりました。

  • 開発者

    CSS の position 属性に sticky と実験的に指定できるようになりました。(Auroraのみ) デモはこちらでご覧になれます。

  • 修正

    ページの自動ロード時に Explore By Touch が動作しないことがある不具合を修正しました。(Bug 963000)

既知の問題

  • 未解決

    Google で最近行われたデザイン変更によって、Firefox など WebKit 以外のブラウザーで問題が生じているところがあります。(Bug 668288)

  • 未解決

    複数のチャンネル (Aurora、Beta など) の Firefox がひとつの端末にインストールされていると、Firefox Sync が期待通りに動作しません。(Bug 761206)
