Android 版 Firefox リリースノート

バージョン — リリース

Firefox に貢献いただいたすべての方に感謝いたします。新たに貢献された方々はこちらからご覧になれます。

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    Android のクリップボードから、編集可能なウェブコンテンツに対してペーストが行えるようになりました

  • 新機能

    ウェブアクセシビリティ: ARIA 1.1 の 'switch' ロールをサポートしました

  • 変更

    SSLv3 通信のうち、安全でないものへのサポートを終了しました

  • 変更

    ホワイトリスト中にあるホストを除き、RC4 が利用できなくなりました。なおこのホワイトリストは一時的なものです

  • 変更

    Sync が端末で利用できるもののうち、最も強固な TLS 設定でのみ動作するようになりました。自身で Sync サーバーをお持ちの方は"こちらをお読みください

  • HTML5

    Fetch API が標準で有効になりました

  • HTML5

    List-style-typeの値に文字列が指定できるようになりました

  • HTML5

    <link rel="preconnect"> が実装されました

  • HTML5

    CSS トランジションとアニメーションのカスケーディングが、現在の仕様のものになりました

  • HTML5

    CSS Scroll Snap Points をサポートしました

  • 修正

    IPv6 の IPv4 へのフォールバック性能が向上しました

  • 修正

    ダウンロード中に HTTP 1.1 での転送が切断されたことを検知すると、ダウンロードが完了したと表示されることがある不具合を修正しました

  • 修正

    いくつかのセキュリティ問題が修正されました
