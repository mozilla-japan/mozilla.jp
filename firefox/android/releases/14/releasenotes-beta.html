Android 版 Firefox リリースノート

バージョン ベータ版 — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    起動やページの読み込みが高速になりました。

  • 新機能

    ユーザーインタフェースを完全に再設計し、新しいスマートスクリーンを備えました。

  • 新機能

    スタートページを再設計し、トップサイト (よく見るページ一覧) を表示するようにしました。

  • 新機能

    Adobe Flash プラグインに対応しました。

  • 新機能

    初期設定では、プラグインコンテンツはタッチしてから読み込まれます。

  • 新機能

    文字サイズの自動拡大によって、可読性を向上させました。

  • 新機能

    パンとズームの処理を再設計し、タッチの応答性を向上させました。

  • 新機能

    Google 検索を SSL 暗号化通信に切り替えました。

  • 変更

    トップサイト、ブックマーク、履歴をスマートロケーションバーから参照しやすくしました。

  • 開発者

    Pointer Lock API を実装しました。

  • 開発者

    ディスプレイをスリープさせないようにする新しい API を実装しました。(Bug 697132)

  • 開発者

    テュルク諸語とギリシャ語のために、CSS プロパティ text-transform、font-variant を 改善しました。

  • 修正

    ページを再読み込みすると文字サイズが拡大されてしまう問題を修正しました。(Bug 628269)

  • 修正

    フォーム入力欄で文字が点滅する場合がある問題を修正しました。(Bug 665374)

  • 修正

    前回のセッション (開いていたタブ) が特定の状況で起動時に復元されない問題を修正しました。(Bug 697858)

  • 修正

    一部の文字が非常に大きく表示される問題を修正しました。(Bug 710808)

  • 修正

    外部アプリケーションからのリンクが正しく開かれない場合がある問題を修正しました。(Bug 711515)
