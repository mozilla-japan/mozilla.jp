バージョン — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

このバージョンのリリースノート

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    タブレット向けにネイティブ UI を実装しました。起動が高速になり、スマートスクリーンも洗練されました。

  • 新機能

    メニューからモバイル向けサイトとデスクトップ向けサイトを切り替えて表示できるようになりました。

  • 新機能

    タブをさらに改良し、スワイプで閉じられるようにしました。

  • 新機能

    ページ内検索に再対応しました。

  • 新機能

    SPDY ネットワークプロトコル v3 に対応しました。

  • 新機能

    Android 標準ブラウザーからブックマークと履歴を取り込めるようになりました。設定画面に項目があります。

  • 変更

    ダウンロードマネージャーに、中止、再開、キャンセル、再試行の各機能を追加しました。

  • 変更

    設定画面でプライベートデータを消去するとき、消去するデータの種類を選択できるようになりました。

  • 開発者

    高精度イベントタイマーを実装しました。

  • 開発者

    CSS word-break プロパティを実装しました。

  • HTML5

    Opus オーディオコーデックに ネイティブ対応 しました。

  • HTML5

    audio/video 要素の played 属性に対応しました。

  • HTML5

    source 要素の media 属性に対応しました。

  • 修正

    テキスト選択に Android の機能を利用するようになりました。(Bug 774938)

  • 修正

    テキストのコピーやペースト機能を再実装しました。(Bug 770928)

  • 修正

    フォームが完全に機能しない問題を修正しました。(Bug 705201)

  • 修正

    Adreno 端末では WebGL が動作しない問題を修正しました。(Bug 736123)

  • 修正

    Android 4.1 (Jelly Bean) で起動時に黒い画面になる問題を修正しました。(Bug 770047)

  • 未解決

    Google で最近行われたデザイン変更によって、Firefox など WebKit 以外のブラウザーで問題が生じているところがあります。(Bug 668288)

  • 未解決

    Android 版 Firefox はシステムの言語設定と同じロケールを使用します。(Bug 695565)

  • 未解決

    文字サイズの自動拡大機能によって、一部のページで文字サイズが不釣り合いになる場合があります。(Bug 707195)

  • 未解決

    端末の向きを変えたときに、意図しない挙動になる場合があります。(Bug 740691)

  • 未解決

    複数のチャンネル (Aurora、Beta など) の Firefox がひとつの端末にインストールされていると、Firefox Sync が期待通りに動作しません。(Bug 761206)

  • 未解決
