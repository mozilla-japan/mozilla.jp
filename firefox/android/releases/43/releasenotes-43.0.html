バージョン — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、Firefox の改良にご協力ください。

  • デスクトップ版
  • Android 版

このバージョンでの変更点

  • 新機能

    未読 / 既読 をリーディングリストパネルで管理できるようになりました

  • 新機能

    タブリスト中で、音声を出力しているタブが確認できるようになりました

  • 新機能

    スクロール、ズーム、点字表示、コンテキストメニューのスクリーンリーダーよる読み上げ、のそれぞれの点でアクセシビリティが向上しました

  • 新機能

    横向きに利用している際に、履歴を分割して表示するようになりました

  • 新機能

    Android の最新デザインにあわせて、ツールバーメニューのデザインを更新しました

  • 新機能

    m4v 形式の動画再生に関する API のサポートを向上しました。

  • 新機能

    URL を含むテキストを、URL を含めて共有するようになりました

  • 修正 いくつかのセキュリティ問題 を修正しました。
