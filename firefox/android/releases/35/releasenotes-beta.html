このバージョンでの変更点

  • 新機能

    WiFi と携帯網のシグナル情報を共有して Mozilla のジオロケーションサービスを向上に貢献する機能が付きました。これはメニューの設定を開き、"Mozilla" にあるデータ欄の貢献するにチェックすることで有効になります。

  • 新機能

    安全な検索のために、Bing 検索に HTTPS を利用するようになりました。

  • 新機能

    ネットワークエラーページに検索用のダイアログが追加されました。

  • 新機能

    Android のダウンロードマネージャーを利用するようになりました。

  • 新機能

    動的なスタイル変更の処理の改善により、応答性が向上しました。

  • 新機能

    HTTP 公開鍵ピンニングが実装されました。暗号化通信を行うサーバーの認証をより安全・確実に行えるようになります。

  • 新機能

    ブルトン語 [br] とエスペラント語 [eo] がロケールに追加されました。

  • HTML5

    JavaScript の "let" の意味が変更されました。

  • HTML5

    Resource Timing API が実装されました。

  • HTML5

    CSS フィルターが標準で有効になりました。

  • HTML5

    Worker から WebSocket が利用できるようになりました。

  • 開発者

    セレクタにマウスオーバーすることで、それにマッチする要素が強調表示されるようになりました。

  • 修正

    拡大、もしくは縮小された画像に必要なリソースが減少しました。

  • 修正

    プライベートブラウジングモードでダウンロードしたファイルも、ダウンロードマネージャーから見られるようになりました。
