まったく新しい Firefox に備えて、心の準備はいいですか。それは高速です。とても高速です。Mozilla の先端研究グループから生まれた最新技術を基にした、完全に刷新されたコアエンジンを搭載し、クリーンでモダンなインターフェイスを兼ね備えたこの新バージョンは、半年前の Firefox より 2 倍以上も高速です。今日は私たちが「Firefox Quantum」と呼び、皆さんがお気に入りのサイトや必要な情報へかつてなく素早くたどり着けるようにすることを主眼において設計した、数回のリリースのうちの 1 回目となります。Windows、Mac、Linux をお使いのデスクトップ上で違いを実感してください。Android 版でも実行速度の改善が行われています。そして Android 版と iOS 版のどちらでも、デスクトップ版のルックアンドフィールを反映した新しい デザインシステム を採用しました。Firefox Quantum の詳細は Mozilla Blog をご覧ください。

Firefox の見た目や使い勝手を高め、高速化する作業は決して小さな努力ではありませんでした。世界中の社員やボランディアが記録的な速さで働き、史上最高の Firefox を作り上げたのです。このバージョンに貢献いただいた 新しい Mozillia コミュニティメンバー 全員にこの場を借りて感謝します。

新機能

  • ページの読み込みを高速化し、より高い安定性を提供するパフォーマンス向上。

  • 最も頻繁に訪れているサイトや Pocket からのおすすめ記事 (現時点では米国、カナダ、ドイツのみ) を含む、改良された新しいタブページなど、インターフェイスの刷新。

  • メディアを再生しているタブが背面へ送られた際に動画のデコーディングを停止するようにしました。動画の再生はタブが前面へ戻ったときに再開されます。音声は影響を受けません。

  • プライベートブラウジング以外でもトラッキング防止を有効化できるオプションを追加しました。

  • プライベートブラウジング中に対応キーボードでプライベートモードを自動的に有効化するようにしました。

  • アドレスバー内の長い URL をスクロール可能にしました。

  • ウォロフ語 (wo) ロケールを追加しました。

修正点

  • 様々な セキュリティ問題 を修正しました。

変更点

  • Android 版 Firefox は今後 WebExtensions API で作られた拡張機能のみに対応し、非対応となった旧式拡張機能を無効化します。背景となった 拡張機能のパフォーマンスやセキュリティを改善する取り組みについてはこちらで説明しています。

開発者

  • Android アプリがカスタマイズされたウェブブラウザーを起動できるようにしました。
