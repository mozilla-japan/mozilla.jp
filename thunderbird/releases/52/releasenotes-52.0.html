Thunderbird リリースノート

バージョン — リリース

このバージョンの変更点と既知の問題を掲載しています。いつものように フィードバックを提供 するか バグを報告 して、製品の改良にご協力ください。

このバージョンでの変更点

  • 新機能

    フォルダーペインツールバーとフォルダービューセレクターを追加しました。(フォルダービューの矢印ボタンを置き換えるものです)

  • 新機能

    Thunderbird からアカウントを削除する際、任意で関連データを削除できるようにしました。

  • 新機能

    Becky! Internet Mail から設定をインポートできるようにしました。

  • 新機能

    メッセージフィルターをコピーできるようにしました。

  • 新機能

    下書き編集時に辞書設定を復元するようにしました。これは Content-Language ヘッダー (RFC 3282) をメッセージに付随して保存することで実現しています。

  • 新機能

    カレンダー: タブ内でイベントの作成と編集を可能にしました。

  • 新機能

    カレンダー: 受信した招待メッセージのカウンター提案処理を追加しました。

  • 新機能

    チャット: Twitter のダイレクトメッセージに対応しました。

  • 新機能

    チャット: Twitter のツイートに「いいね」できるようにしました。

  • 新機能

    チャット: XMPP で SASL SCRAM 認証機構に対応しました。

  • 新機能

    チャット: Jabber/XMPP メッセージカーボン (XEP-280) に対応しました。

  • 変更点

    重要: 日本語版で、既定の文字エンコーディングを ISO-2022-JP から Unicode (UTF-8) へ変更しました。現在では各種メールクライアントにおいて Unicode 対応が改善されており、問題が生じる可能性は少ないと思われますが、何らかの理由で設定を戻す必要がある場合は、オプション画面の [表示] > [書式] > [フォントと配色] > [詳細設定] を開き、エンコーディングを変更してください。

  • 変更点

    重要: メッセージ編集ウィンドウにおける画像の追加方法が変わりました。従来は他のメッセージの一部や OS 上のファイルへの参照でしたが、今後は data URI として追加されるようになります。これにより、MS Office や LibreOffice といったオフィスパッケージとの相互運用性が向上します。インターネット上の場所からリンクされた画像は、自動的にダウンロードされてメッセージへ添付されることがなくなります。この挙動は、各画像のプロパティダイアログで個別に、あるいは mail.compose.attach_http_images 設定で全体的に変更することもできます。

  • 変更点

    「通信相手」列をすべての新規フォルダーに適用するようにしました。この挙動は mail.threadpane.use_correspondents 設定で無効化できます。

  • 変更点

    メーリングリストへの返信時に、From ヘッダー内のアドレスに対して送信し、Reply-to ヘッダーを無視するようにしました。

  • 変更点

    Linux で音声の再生に PulseAudio が必要となりました。

  • 変更点

    送信形式がプレーンテキストのみに切り替えられた際に、書式ツールバーを表示したままにしました。

  • 変更点

    外付け端末から読み込まれた IMAP フォルダー内のメッセージに対して初期設定でフィルターが適用されるようになりました。

  • 変更点

    4 GB 以上の mbox ストレージに保存されたフォルダーを警告なしに読み込むようにしました。(mailnews.allowMboxOver4GB 設定が false に設定されていない限り)

  • 変更点

    IMAP のキャッシュに Mozilla の最新キャッシュ技術を採用しました。

  • 変更点

    チャット: Yahoo! Messenger 対応を削除しました。(Yahoo! が対応を廃止したためです)

  • 修正

    IMAP フォルダーの名前変更後や削除後にメッセージプレビューペインが動作しなくなる問題を修正しました。

  • 修正

    段落形式での編集時に、Shift+Enter キーを押しても次の行へカーソルが移動しないことがある問題を修正しました。

  • 修正

    段落形式でのメッセージ編集時に見られた様々な問題を修正しました。

  • 修正

    引用文として貼り付けが動作しないことがある問題を修正しました。

  • 修正

    プレーンテキスト形式での返信時に、長い行が適切に折り返されない問題を修正しました。

  • 修正

    段落モード時に署名の前に不要な空白が入る問題を修正しました。

  • 修正

    編集画面で添付ファイルを追加できなかった場合に、エラーではなく「添付しています...」というメッセージが表示され続ける問題を修正しました。

  • 修正

    返信メッセージの文字エンコーディングが正しくない場合がある (最後に表示されたメッセージのエンコーディングが使用されてしまう) 問題を修正しました。

  • 修正

    メッセージの表示、返信、あるいは転送されたメッセージの文字エンコーディングが正しくない場合がある (添付ファイルのエンコーディングが使用されてしまう) 問題を修正しました。

  • 修正

    保存された下書きの送信形式 (自動判別、プレーンテキスト、HTML、両方) が維持されない問題を修正しました。

  • 修正

    自分のメールに対して返信する際に、適切な差出人情報が選択されない問題を修正しました。

  • 修正

    IMAP メッセージが部分的にキャッシュされてしまう問題を修正しました。

  • 修正

    エスケープされた非 ASCII (国際) 文字を含むリンクがクリックできない問題を修正しました。

  • 修正

    カレンダー: タイムゾーンが「現地時間」に指定されたイベントのアラートが UTC 時刻に表示される問題を修正しました。

  • 修正

    チャット: XMPP のリソース競合を修正しました。
