Thunderbird リリースノート

バージョン 2 Beta 1

Thunderbird 2 Beta 1 は、次世代 Thunderbird 電子メールクライアントの開発者向けプレビューリリースで、テスト目的にのみ限って 提供されます。Thunderbird 2 Beta 1 はテストコミュニティを対象にしています。

このリリースノートには、Thunderbird 2 Beta 1 に関する 新機能、ダウンロード、インストール方法、既知の問題、ユーザーサポート / フィードバック方法 が載っています。Bugzilla にバグを報告する前に、このリリースノートと バグの登録方法 をよくお読みください。

こちらにある フィードバック欄 からフィードバックをお寄せくださるか、MozillaZine がホストする Mozilla Thunderbird フォーラムにお越しください。

本開発マイルストーンの新機能

Thunderbird 2 Beta 1 には、いろいろ試すことができる要素がたくさん詰まっています：

  • メッセージタグ： 電子メールの整理のために独自のタグを作成できるようになりました。メッセージには好きなだけタグを割り当てることができるようになりました。タグは検索フォルダーやメールビューと連携させることができ、電子メールの整理が簡単にできるようになっています。
  • テーマ：Thunderbird 2 のテーマおよびユーザーインターフェースがアップデートされ、利便性が向上しています。
  • セッション履歴ナビゲーション： 進む・戻るボタンからメッセージ履歴をナビゲートできるようになりました。
  • フォルダービュー： フォルダーペインをカスタマイズして、お気に入りや未読フォルダー、最近のフォルダーを表示できるようになりました。
  • 拡張機能へのより良い対応： 拡張機能を用いて、メールデータベースにカスタムのメッセージデータを追加できるだけでなく、カスタムコラムをメッセージリスト領域に追加できるようになりました。
  • 改良されたオフライン管理機能： Thunderbird (Linux および Windows) では、その時点のネットワーク状態に応じて、自動的にオフライン状態を調節できるようになりました。
  • 改良された新着メール通知機能： 新しいメール通知機能には、件名、送信者およびメッセージ本文といった情報が含まれるようになりました。
  • フォルダーの要約ポップアップ： 新着メッセージが含まれるフォルダーにマウスをおくと、そのフォルダー内にある新着メッセージの要約を確認できるようになりました。
  • 検索フォルダーのパフォーマンス： 検索フォルダーの検索結果がキャッシュされるようになり、フォルダーの読み込みパフォーマンスが向上しました。
  • Find As You Type
  • 改良されたファイリングツール： 最近利用したフォルダーへフォルダーを移動・コピーするための、最近利用したメニュー項目 (訳注：自信ありません) 再び移動・コピーする機能
  • 拡張機能システムのアップデート： 拡張機能システムはアップデートされ、より充実したセキュリティ機能を提供し、拡張機能をより簡単にローカライズできるようになりました。
  • 新しい Windows インストーラ： Nullsoft Scriptable Install System を採用しました。これまで長い間解決できなかった問題が、新しい Windows インストーラによって解消されました。
  • MacOS X でのユニバーサルバイナリ化

Rumbling Edge に 重要なバグ修正 に関するより詳しい一覧があります。

ダウンロードとインストール

必要システム構成

インストールの前に、お使いのコンピューターが 必要システム構成 を満たしているか確認してください。

Thunderbird 2 Beta 1 のダウンロード

Mozilla Foundation では、Windows、Linux、そして Mac OS X に対応した Thunderbird を提供しています。

Windows： インストーラ

Linux： GTK2 + XFT (インストーラではありません)

Mac OS X： 圧縮されたディスクイメージ

mozilla.org によって提供されていない、他のシステム・言語向けのビルドについては、このページの最後にある「提供されたビルド」の項目をご覧ください。

Thunderbird のインストール

Thunderbird のインストーラ、あるいは圧縮アーカイブをダウンロードしたら、次の指示に従ってインストールしてください：

Windows

Thunderbird Setup 2.0 Beta 1.exe インストーラをダブルクリックして、インストールします。

MacOS X

Thunderbird 2 Beta 1.dmg ファイルをダウンロードしたら、Thunderbird のディスクイメージをダブルクリックして Finder に開き、Thunderbird のアプリケーションアイコンをハードディスクにドラッグしてください。アプリケーションを Dock に表示したい場合は、アイコンを Dock にドラッグしてください。

その他のプラットフォーム

圧縮されたアーカイブを解凍して、thunderbird を実行します。

Thunderbird のアンインストール

Thunderbird をアンインストールするには、次の指示に従ってください：

Windows

スタート メニューから コントロールパネル を選択します。コントロールパネルが表示されたら、プグラムの追加と削除 をダブルクリックします。一覧から 「Mozilla Thunderbird (2.0b1)」 を探して、削除 ボタンをクリックしてアンインストールします。

MacOS X

Thunderbird アプリケーションをゴミ箱にドラッグします。

その他

thunderbird フォルダーを削除します。

以上のガイドではプロファイルデータはそのまま残ります。将来 Thunderbird をインストールした際に再び使えるようにするためです。プロファイルフォルダーを削除したい場合は、プロファイルフォルダー のセクションで示されているフォルダーを削除してください。プロファイルフォルダーを削除すると、電子メール、アドレス帳データ、設定その他の情報が削除され、再び Thunderbird を使用する際にはまっさらなプロファイルで初めから設定し直す必要があります。

その他の情報

プロファイルフォルダー

Thunderbird は次の場所のいずれかにユーザーデータを保存します：

Windows 2000、XP Documents and Settings\<ユーザー名>\Application Data\Thunderbird
Windows NT WINNT\Profiles\<ユーザー名>\Application Data\Thunderbird
Windows 98、ME Windows\Application Data\Thunderbird
MacOS X ~/Library/Thunderbird
Linux および Unix システム ~/.thunderbird

拡張機能とテーマ

Thunderbird 1.5 環境下でインストールされた拡張機能には、互換性が無く、Thunderbird 2 Beta 1 で利用するためにアップデートが必要となるものもあります。なにか問題が見つかった場合は、その拡張機能の管理者にご連絡ください。Thunderbird 2 Beta 1 をインストールすると、すべての拡張機能とテーマは、Thunderbird が次のいずれかの判断をするまで利用できなくなります： a) Thunderbird 2 Beta 1 リリースと互換性がある、b) 互換性のある新しいバージョンが利用できる。

既知の問題

Thunderbird 2 Beta 1 で確認されている問題の一部を掲載します。新しいバグを報告する前に、かならずお読みになるとともに、定期的に確認してください (リリースに新しいバグが見つかるたびに更新します) 。　

すべてのシステム

ロケール
Thunderbird 2 Beta 1 は en-US でのみ提供されます。ローカライズ版は Beta 2 で登場します。
スペルチェック：
標準の辞書は英語のみです。英語以外の辞書は、addons.mozilla.org から拡張機能としてインストールできます。
拡張機能：
Thunderbird の過去のバージョンから Thunderbird 2 Beta 1 へと移行する際、一部の拡張機能やテーマが無効化される可能性があります。上記 「拡張機能とテーマ」 をご覧ください。
LDAP over SSL
SSL を経由した LDAP 参照は、Thunderbird 2 Beta 1 では機能しません。この問題は Beta 2 までに解決する予定です。
共通受信トレイ：
共通受信トレイと POP3 アカウントの受信箱を行き来して利用する場合は、必ず一方のフォルダーにメッセージのコピーを残しておいてください。

 

よくある質問

  1. どのような協力ができますか？

    できるだけ多くの人の目に触れるようにしたいと考えています。できるだけ多くの友人や家族、同僚の皆さんに Thunderbird を奨めてください。学生さんの場合は、学校で配布してください。 Slashdot やその他のニュースサイトにリリースについて投稿してください。ご自分のブログに投稿してください。インターネットでは大量配布が可能です。Kazaa が良い例です。

  2. なぜ私が送ったメールに返事をくれないのですか？

    フォーラム をご利用ください。Thunderbird チームは常時このフォーラムをチェックしています。私たちは毎日多くのメールを受け取っているので、あなたが送ったメールは見過ごされる可能性もあります。

  3. アドオン (拡張機能) はどこで手に入りますか？

    拡張機能 と テーマ は Mozilla Update からダウンロードできます。

  4. 誰が Thunderbird の開発をしているのですか？

    非常に多くの人が関わっています。Thunderbird に貢献している人たちの一覧を見るには、［ヘルプ］ メニューの ［Mozilla Thunderbird について］ を開いて「クレジット」をご覧ください。
