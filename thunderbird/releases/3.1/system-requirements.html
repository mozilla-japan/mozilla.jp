Thunderbird システム要件

Thunderbird 3.1

Windows

オペレーティングシステム (OS)

  • Windows 2000
  • Windows XP
  • Windows Server 2003
  • Windows Vista
  • Windows 7

最小ハードウェア構成

  • Pentium 233 MHz プロセッサー (Pentium 500 MHz 以上を推奨)
  • Windows 7、Vista、XP: 786 MB の RAM (1 GB 以上を推奨)
  • Windows 2000: 256 MB の RAM (512 MB 以上を推奨)
  • 52 MB のハードディスク空き容量

Mac

オペレーティングシステム (OS)

  • Mac OS X v10.4 (Tiger)
  • Mac OS X v10.5 (Leopard)
  • Mac OS X v10.6 (Snow Leopard)

最小ハードウェア構成

  • Intel x86 または PowerPC G3/G4/G5 プロセッサーを搭載した Macintosh コンピューター
  • 256 MB の RAM (512 MB 以上を推奨)
  • 200 MB のハードディスク空き容量

Linux

ソフトウェア要件

注: 各 Linux ディストリビューターは、それぞれのディストリビューションに対して、システム要件が異なるパッケージを提供している場合もあります。

Thunderbird 3.1 は、以下のライブラリもしくはパッケージがインストールされていない環境では正常に動作しません。

  • GTK+ 2.10 以上
  • Glib 2.12 以上
  • Pango 1.14 以上
  • X.Org 1.0 以上

また、快適に利用するため、以下のライブラリもしくはパッケージを推奨します。

  • NetworkManager 0.7 以上
  • DBus 1.0 以上
  • HAL 0.5.8 以上
  • GNOME 2.16 以上
