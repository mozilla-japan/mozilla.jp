法人向けサポートサービスのご案内

法人利用の現場でも Firefox を安心してお使いいただけるよう、Mozilla Japan およびサポートパートナー各社が提供する「法人向けサポートサービス」をご利用ください。

法人向けサポートサービス

Mozilla Japan では、企業および行政、教育機関などの皆様向けに、Firefox の利用をお勧めしています。一括導入や集中管理、カスタマイズに際して有償でのサポートサービスが必要な場合、Mozilla Japan による法人向けサポートサービスもしくは Mozilla Japan の サポートパートナー による各種サービスをご利用いただけます。

Mozilla Japan による法人向けサポートサービスについてのお申し込み、お問い合わせについては Mozilla Firefox 法人向けサポートサービス担当 にお問い合わせください。

サポート内容

Mozilla Japan による Firefox 法人向けサポートサービスでは、Firefox に関する次のお問い合わせに電子メールにて対応いたします。

  • 使用方法に関するお問い合わせ対応
  • 機能に関するお問い合わせ対応
  • 不具合に関するお問い合わせ対応
  • 動作に係るソースコードレベルの調査
  • 不具合解決および機能追加のためのアドオン開発

サポート内容、サービスメニューについては サポートサービス紹介 (PDF) をご覧ください。ご不明な点などあれば Mozilla Firefox 法人向けサポートサービス担当 までお問い合わせください。

カスタマイズ

Firefox の法人利用の現場でよく見られる 一般的なカスタマイズ は、集中管理の設定やすでに公開されているアドオンの利用で実現可能です。個別のニーズにマッチした専用のカスタマイズや集中管理をご希望の場合は、有償の法人向けサポートサービスにて、ご要望に応じた様々なカスタマイズに対応いたします。

  • バージョンアップの集中管理
  • 指定のアドオン以外をインストール禁止
  • パスワードやフォーム入力を記憶させない
  • 位置情報やカメラへのアクセスを禁止
  • バックグラウンド通信の禁止
  • 複数バージョンの Firefox をサイト別に切り替えて利用

その他、柔軟にカスタマイズが可能です。

サポートパートナー

以下のパートナー各社が法人向け有償サポートを提供しています。サポート内容、価格面などの詳細につきましては、各パートナーへ直接お問い合わせください。どのパートナーを選んだらいいか迷われるときは、Mozilla Japan にご相談ください。

株式会社クリアコード

www.clear-code.com

業務内容

株式会社クリアコードはフリーソフトウェア開発者が中心となって設立した会社で、フリーソフトウェアの普及と発展に貢献することを目標としています。フリーソフトウェアの開発を柱に、フリーソフトウェアのカスタマイズや品質改善、障害解析、フリーソフトウェアを活用したシステム構築を事業として展開中です。

サポートパートナーになったきっかけ

クリアコードには Firefox のアドオン開発者や Mozilla のテクノロジーを活用したブラウザーの開発者がいたことから、Mozilla Japan とは会社設立当初から技術的な面で協力関係にあり、それがきっかけでサポートパートナーになりました。オープンソース業界の情報交換や開発者の啓蒙、教育などでも協力していて、Mozilla 製品の開発者向けセミナーで弊社スタッフが講師を行ったこともあります。

具体的なサポート体制

サポートパートナーとして以下のサービスを展開しています。

  1. 導入支援およびサポート: 企業・官公庁・学校向けに Thunderbird、Firefox の導入支援およびサポートを行います。Thunderbird の導入においては、組織にあわせた Thunderbird のカスタマイズから Thunderbird と連携した迷惑メール対策システムの導入、Active Directory との連携など組織のメールシステム全般について支援します。
  2. アドオン、プラグイン開発: Firefox 用の動画再生プラグインや、ツールバー、アドオンを開発します。
  3. 組み込み機器向けソフトウェア開発: Mozilla テクノロジーを活用した各種組込み機器向けブラウザーの開発などを行います。

サポートパートナーになったきっかけ

Firefox をはじめとするモダンブラウザーは、高速に、美しく小気味よく動作するシステムを実現する上で、今や必要不可欠な存在です。弊社では、スピーディなシステム開発と、より使いやすいシステムの実現のために、従来より積極的にお客様に Firefox の導入を推進しております。一緒にお仕事をさせていただくことで、より多くのお客様にモダンブラウザーがもたらす価値を我々の価値観で提供することができ、またそれが Firefox のエンタープライズ市場における普及促進の一助になればと思い参加させていただきました。

具体的なサポート体制

Internet Explorer に依存したシステムを考慮した Firefox への移行、導入支援、運用サポートをはじめ、Firefox をターゲットとしたウェブシステム開発を行っております。すでに大手建設不動産業のお客様に Firefox とアドオンをパッケージ化し、弊社が導入した営業支援システムのフロントエンドとして活用いただいております。各サービスについての詳細は、弊社ウェブページよりお問い合わせください。

ミラクル・リナックス株式会社

www.miraclelinux.com

業務内容

ミラクル・リナックス株式会社は、オンプレミス、クラウド、組込みのあらゆる領域で活用できるエンタープライズ Linux OS とサポートサービスを提供しています。Linux のカーネル技術をさまざまな分野に応用し、デジタルサイネージや IoT 分野にも組込み Linux の実績を伸ばしています。

さらに、オープンソースベースの企業向け統合監視ツールやバックアップソフトウェアを開発しているほか、統合運用ソフトウェアを開発し、オープンソースとしてコミュニティに公開するなど、オープンソースの活性化に向けた取組みも積極的に行っています。

サポートパートナーになったきっかけ

ミラクル・リナックスは、企業や行政、教育機関など、法人で利用するブラウザーとして、Mozilla Firefox ブラウザーをおすすめしています。

Linux OS をはじめとするミラクル・リナックスの製品・サービスは、通信事業や公共、金融事業などのエンタープライズ領域で幅広い採用実績があり、Mozilla Firefox のメリットを享受できる法人のお客様との親和性が高いことから、サポートパートナーになりました。

具体的なサポート体制

Mozilla Japan が提供する「Mozilla Firefox 法人向けサポートサービス」の販売代理店として、法人のお客様の Mozilla Firefox 導入・活用を推進します。またセキュリティ強化の観点から Mozilla Firefox を活用した社内システム構築をご提案します。

さらに今後、組込み Linux 分野において Firefox (Gecko) を採用されるお客様に対する支援サービスの提供を予定しています。ミラクル・リナックスの持つ Linux カーネルの技術によって、組込み領域における Mozilla Technology の普及に貢献していきます。

株式会社 Digital Stacks

www.digitalstacks.net

業務内容

株式会社 Digital Stacks は各種ウェブサービス (ウェブ制作・開発、広告、調査・分析) をトータルに提供しているウェブコンサルティングファームです。国内のみならず中国を中心としたアジア市場でのサポート実績も豊富で、地域を超えたウェブビジネスの包括的ソリューションを提供しており、グローバルに企業のウェブ戦略支援を行っています。

サポートパートナーになったきっかけ

1990 年代中頃から、インターネット・ウェブに関するコンサルティング事業を行う中で Mozilla の前身となっている Netscape のブラウザーをターゲットに、多くの著名ウェブサイトを開発しておりました。Netscape から Mozilla に移行後も、ウェブ産業拡大の原動力としてのブラウザーを支援することが業界の拡大につながると考え、サポートパートナーになりました。Firefox を使用したオリジナルブラウザーのデザイン・開発等も行いました。
